tclib
=====

Trinity Core compatible client library

I used this library for make [web chat](https://www.corechat.eu/).

Installation
============
    git clone https://github.com/meggie-true/tclib.git
    cd tclib
    python setup.py install

Supported server versions
=========================
* 4.3.4
* 4.2.2
* 4.0.6a
* 3.3.5a
* 3.3.3a
* 3.3.2
* 3.3.0a
* 3.2.2a
* 3.1.3
* 2.4.3
* 1.12.3
* 1.12.2
* 1.12.1
